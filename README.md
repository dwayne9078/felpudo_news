## Run Locally

Clone the project

```bash
  git clone https://github.com/dwayne9078/felpudo_news.git
```

Go to the project directory

```bash
  cd felpudo_news
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
